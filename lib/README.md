# English chat Flutter Codebase

A premium, luxury real-time communication mobile and web application engineered specifically for learning and practicing English, complete with a clean architecture, instant translation service, real-time presence indicators, and peer calling.

## Key Configurations

- **Application Name**: English chat
- **Android Package Identifier**: `com.englishchat.app`
- **Firebase Project ID**: `english-chat-app-prod`

## Implemented Modules

1. **AI-Powered Translations**: Fast translation of long-press selected messages into your preferred language using Google Translation services.
2. **Online/Offline Status & Presence**: Integrated with `WidgetsBindingObserver` to update real-time presence status in Cloud Firestore.
3. **Settings & Custom Profiling**: High-fidelity user profiles specifying native translation languages, personal bios, and custom avatars.
4. **Interactive Private & Group Chat**: Integrated with message actions (reply, delete), read receipts, and media attachments.
